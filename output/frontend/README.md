# BANKACCT Frontend

## Setup

```bash
npm install
# If production-only installs are default in your environment:
npm install --include=dev
```

## Run

```bash
npm start
```

## Scheduled Tasks

Use the `Scheduled Tasks` navigation entry (`/scheduled-tasks`) to create cron-based jobs, run them on demand,
and view execution logs.

## Tests

```bash
npm test
npm run e2e
```
